# Pilot Kickoff Checklist

## Readiness

- [ ] Week 4 decision is `Go`.
- [ ] Signed artifact metadata recorded (run URL, artifact ID, SHA256, signing identity, notarization status).
- [ ] Support runbook reviewed by PM and Engineering owners.
- [ ] Rollback runbook reviewed and acknowledged.

## Distribution

- [ ] Participant roster finalized.
- [ ] Install instructions shared with participants.
- [ ] Known-good artifact link and checksum shared.
- [ ] Communication channel and support hours shared.

## First-Day Controls

- [ ] Incident log initialized (`docs/reports/PILOT_INCIDENT_LOG.md`).
- [ ] Triage cadence confirmed.
- [ ] Escalation owner on call.
